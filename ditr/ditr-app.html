Транскрибатор DITR

Загрузите аудио-файл, чтобы получить дословную стенограмму без перевода. Приложение использует модель Whisper через защищённый прокси и помогает контролировать арабские цитаты из Корана, хадисов и поэтов.

Файл не выбран
Поддерживаются форматы: mp3, wav, m4a, ogg, webm и другие. Можно перетащить файл в это окно.

Температура модели фиксирована на 0, чтобы не искажать цитаты. При монолингвальной записи выберите нужный язык вручную.

Проверьте ключевые фрагменты

Эти сегменты содержат арабский текст или кавычки. Сверьтесь с оригиналом, чтобы убедиться, что стенограмма не искажена.

    Версия 2.0 • Расшифровка выполняется через Cloudflare Worker с моделью large-v3 по конечной точке /v1/audio/transcriptions.
